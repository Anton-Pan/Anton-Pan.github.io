TicketRebel
[ 2024 - Present ]
I am devloping a concert ticketing website, focused on local, small-scale events, with emphasis on concerts. Unique features will include instant payment to event organizers upon ticket purchase, and a self-contained ticket reselling marketplace, designed to prevent scalping and minimize unused tickets.
My focus on the project at the moment is designing an event sourcing-based database model to properly manage user, ticket and event data. Additionally, I am doing market research through interviewing event organizers and analyzing survey data send in by local eventgoers.
Made using:
React,
Material UI for UI/UX,
Koa for content routing,
AJV for data validation,
Swagger/OAS3 for API design,
Stripe for payments, and
PostgreSQL for data storage and managment.
Last Minute Massage
[ 2021-2023 ]
This was one of my first big projects - it was a website that displayed an interactive Google Maps API-based page populated with massage parlors and information on their available time slots and services.
Made using:
HTML/CSS for frontend,
NestJS for backend, and
a combination of Selenium webscraping and JaneApp API calls for data collection.
