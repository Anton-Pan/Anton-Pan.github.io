Space Explorer Hub
[ 2025 ]
Space Exporer Hub is an interactive GlobeGL-based site that visualizes rocket launches and real-time, TLE-propagated satellite orbits, where users can filter rocket launches by month-year ranges and view detailed statistics about each one, and can center and time-seek the orbiting satellites.
We approached the project using the Agile Scrum methodology. As project manager, I hosted weekly Scrums to check in with the rest of the team, resolve conflicts and assign duties.
Made using:
React,
Material UI for UI/UX,
react-globe.gl for 3D globe visualization,
satellite.js for satellite orbit calculations, and the
Launch Library II and CELESTRAK APIS for content/data.
The completed project is hosted on GitHub Pages, and the source code can be viewed on GitHub.
Geofitti
[ 2025 - Present ]
Geofitti is a GIS web app that allows users to pin content (user-drawn SVG art, text-based notes or reviews, etc.) onto an interactive map of Earth, featuring 'like'-count leaderboards, limited-lifetime content pins, and crowdsourced moderation.
I am currently working on implementing content creation, including a FabricJS canvas to enable on-screen, in-app drawing, and planning data-storage strategies.
Made using:
GeoServer for map tile serving.
PostGIS for geospatial data storage.
OpenLayers for interactive mapping,
Koa for proxying the map tile server and serving assets, and
Material UI for UI/UX.
TicketRebel
[ 2024 - Present ]
TicketRebel is a WIP concert ticketing web app focused on local, small-scale events, with emphasis on concerts.
Unique features will include instant payment to event organizers upon ticket purchase, and a self-contained ticket reselling marketplace, designed to prevent scalping and minimize unused tickets.
Made using:
React,
Material UI for UI/UX,
Koa for content routing,
AJV for data validation,
Swagger/OAS3 for API design,
Stripe for payments, and
PostgreSQL for data storage and managment.
Last Minute Massage
[ 2021-2023 ]
Last Minute Massage was my first big project - it was a website that displayed an interactive Google Maps API-based page populated with massage parlors and information on their available time slots and services.
Made using:
HTML/CSS,
NestJS for backend, and
a combination of Selenium webscraping and JaneApp API calls for data collection.
